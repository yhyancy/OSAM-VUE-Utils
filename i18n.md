# vue.js国际化vue-i18n插件的使用问题，在模版文本、组件方法、jsf方法里的使用
注：properties文件参照OSIAMResources.properties

## 1.在文本里使用{{$t("xxx")}}
```
<span>{{$t("properties.code")}}</span>
```
## 2.在组件方法里使用$t('xxx')
```
<md-input-item :placeholder="$t('properties.code')">
```
## 3.在js方法里使用：this.$i18n.t('xxx')
```
this.$i18n.t('properties.code')
```
## 4.如果是在请求后返回的方法里使用需要在上面先定义个变量，var this_ = this; 然后里面使用this_才能正常使用
```
Toast.info(this_.$i18n.t('properties.code'));
```
## 5.如果properties中含有{0}
```
<div v-html="$t('properties.code').replace('{0}',this.name)"></div>
<p v-html="$t('message.500.osi',{0:'我是测试'})"></p>

```
## 6.如果properties中含有.  eg:entitlementsummary.section2.'field1.value3'
```
<el-checkbox :label="$t('entitlementsummary.section2'+`['field1.value3']`)" name="type"></el-checkbox>

```
## 7.引用utils文件写法,不需要import utils,直接this.$Utils.XXXX
```
data () {
    return {
        accesstype: this.$Utils.usage
    }
}

```
